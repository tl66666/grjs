01

AI Agent Career OS · Self-built Runtime

职途 AI Agent

面向真实求职流程的本地优先 Web 系统，把职业目标、简历、JD、面试、机会和复盘行动串成六步闭环。Agent 不是独立聊天框，而是能读取当前业务上下文、调用 22 个受控工具、结合分层记忆把建议推进到可验证的下一步。

Agent 主线：确定性任务始终本地优先；没有 API Key 也能诊断进度、选择简历、生成草稿和定制面试题。配置模型后，再增强完整简历改写与开放表达；所有写入仍遵循“提议 → 预览 → 用户确认 → 领域服务执行”。

PythonFlaskSQLiteSQLite FTS5JSON SchemaPlaywrightTool-Calling Runtime22 ToolsLayered MemoryChart.jsWeb Speech API
  • 自研有界 Tool-Calling Runtime，由 ContextBuilder、MemoryStore、Orchestrator 和 ToolRegistry 协同控制上下文、记忆、工具预算与结果校验。
  • 注册 22 个求职工具，覆盖求职诊断、简历与 JD 匹配、定制面试题、训练复盘、机会跟进和行动优先级生成。
  • 六层记忆支持跨页面恢复上下文；写操作遵循“提议 → 预览 → 用户确认 → 领域服务执行 → 幂等回执”。
  • 产品入口采用“右侧求职 Agent + 求职指挥台”分工：前者负责对话与工具执行，后者汇总待确认操作和机会进展；面试训练中的教练反馈只服务具体练习复盘。
  • 将模型增强与本地执行分层：GLM / DeepSeek / Kimi 用于开放表达，离线核心流程保持可用；支持 SQLite 迁移、Edge/Chrome 和 Windows 一键启动。
