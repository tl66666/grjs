# 使用与素材说明

本仓库用于个人学习、作品集展示和求职展示。

## 简历文件

`唐乐-简历.pdf` 是网站“下载简历”按钮使用的正式文件名。为了保证线上链接稳定，后续更新简历时应继续覆盖这个文件名，不建议在页面中使用带版本或导出平台后缀的文件名。

## 素材说明

仓库中的图片、视频、项目截图、二维码和页面文案仅用于展示作者个人作品与项目实践。部分视觉素材来自作者项目资源、AI 生成素材或项目展示素材整理。

如后续用于正式商业发布、公开模板分发或二次使用，应重新确认所有图片、视频、字体和图标素材的授权范围。

## 隐私说明

仓库中不应提交个人账号密钥、云服务密钥、后台管理口令、私有环境变量或未脱敏的用户数据。简历中的联系方式为作者主动公开的求职联系信息。

## 发布说明

本项目通过 GitHub Pages 发布。线上页面可能存在缓存，推送后若未立即更新，可以等待 GitHub Pages 自动构建完成后再刷新查看。
